# Clipton プライバシーポリシー

最終更新日: 2026-05-16

Clipton は Windows 向けのローカル動作型クリップボード管理アプリです。

## Clipton が扱うデータ

Clipton は、直近のクリップボード履歴を表示し、選択した項目を貼り付けるためにクリップボード内容を読み取ります。クリップボード内容には、テキスト、リッチテキスト、HTML、画像、他アプリでコピーされたファイルパスが含まれる場合があります。

## 保存

現在のバージョンでは、クリップボード履歴は Clipton の実行中のみメモリ上に保持されます。設定と登録単語はユーザーのアプリケーションデータフォルダーにローカル保存されます。

Clipton は、クリップボード内容、設定、登録単語、利用状況データをサーバーへアップロードしません。

## ネットワークとテレメトリ

現在のバージョンには、クラウド同期、分析、テレメトリ、クラッシュレポート、広告 SDK、第三者へのデータ共有はありません。

## ユーザーによる制御

ユーザーはアプリ画面からクリップボード履歴を消去できます。Windows 起動時の開始は、アプリ設定または Windows のスタートアップ設定から無効にできます。

## 将来の変更

永続履歴、同期、テレメトリ、オンライン機能を後から追加する場合は、必要に応じてオプトイン方式にし、リリース前に本ポリシーを更新します。
