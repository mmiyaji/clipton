# Clipton プライバシーポリシー

最終更新日: 2026-07-03

Clipton は Windows 向けのローカル動作型クリップボード管理アプリです。

## Clipton が扱うデータ

Clipton は、直近のクリップボード履歴を表示し、対応する項目をプレビューし、選択した項目を貼り付けるためにクリップボード内容を読み取ります。クリップボード内容には、テキスト、リッチテキスト、HTML、画像、他アプリでコピーされたファイルパスが含まれる場合があります。

## 保存

現在のバージョンでは、クリップボード履歴は既定でユーザーのアプリケーションデータフォルダーにローカル保存され、履歴ペイロードは Windows のユーザー単位 DPAPI で保護されます。登録単語もローカルに保存され、暗号化されます。設定、ローカル画像サムネイル、一時貼り付けファイルもユーザーの端末上に保存されます。ユーザーは設定から暗号化履歴を無効にできます。

ファイルとして貼り付けるための一時ファイルと外部画像プレビュー用の一時ファイルには、クリップボード内容が暗号化されない形で含まれる場合があります。Clipton は貼り付け用ファイルを約30分後、画像プレビュー用ファイルを約10分後に削除するよう最善努力で処理し、起動時にも古い一時ファイルを削除します。

Clipton は、クリップボード内容、設定、登録単語、利用状況データをサーバーへアップロードしません。

## ネットワークとテレメトリ

現在のバージョンには、クラウド同期、分析、テレメトリ、クラッシュレポート、広告 SDK、第三者へのデータ共有はありません。

## ユーザーによる制御

ユーザーは記録の一時停止、Delete キーによる履歴項目の個別削除、アプリ画面からの履歴消去、ローカルデータのエクスポート/インポート、履歴と登録単語を表示する前の PIN 要求、暗号化ローカル履歴の無効化を行えます。Windows 起動時の開始は、アプリ設定または Windows のスタートアップ設定から無効にできます。

## 将来の変更

永続履歴、同期、テレメトリ、オンライン機能を後から追加する場合は、必要に応じてオプトイン方式にし、リリース前に本ポリシーを更新します。
