# Clipton 利用規約

最終更新日: 2026-05-16

Clipton は Windows 向けのローカル動作型クリップボード生産性ツールとして提供されます。

ユーザーは、保持、貼り付け、削除するクリップボード内容を自ら判断する責任を負います。クリップボード内容には、機密情報、認証情報、個人情報、第三者のコンテンツが含まれる場合があります。

法令で認められる最大限の範囲で、Clipton は保証なく提供されます。コピーされた内容、誤貼り付け、履歴削除、Windows クリップボードと連携する第三者アプリケーションにより生じた損害について、メンテナーは責任を負いません。

ユーザーは、適用される法令、勤務先のポリシー、第三者の権利を遵守して Clipton を利用する必要があります。
