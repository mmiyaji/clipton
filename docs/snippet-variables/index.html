Clipton Snippet Variables

Last updated: 2026-05-31

日本語

登録単語の本文に {{date}} のような変数を書くと、登録単語を貼り付ける瞬間の値に展開されます。通常のクリップボード履歴貼り付けでは展開されません。

基本変数

変数内容例
{{date}}現在の日付2026-05-31
{{time}}現在の時刻09:08
{{datetime}}現在の日付と時刻2026-05-31 09:08
{{now}}{{datetime}} と同じ2026-05-31 09:08
{{isodate}}ISO形式の日付2026-05-31
{{isodatetime}}ISO 8601形式の日時2026-05-31T09:08:07.0000000+09:00
{{year}}年2026
{{month}}月05
{{day}}日31
{{weekday}}曜日日曜日
{{unix}}Unix時刻1780199287
{{uuid}} / {{guid}}新しいUUID2d0b45d0-6b9e-4d2c-a8dc-...
{{br}} / {{newline}}改行複数行テンプレート用

日時フォーマット

{{date}}, {{time}}, {{datetime}}, {{now}}, {{weekday}} は .NET の日時フォーマットを指定できます。

{{date:yyyy/MM/dd}}
{{date("yyyyMMdd")}}
{{time:HH:mm:ss}}
{{datetime:yyyy-MM-dd HH:mm:ss}}
{{weekday:ddd}}

例

本日 {{date:yyyy/MM/dd}} {{time}} 時点で確認しました。
受付ID: {{uuid}}
1行目{{br}}2行目

未対応の変数は変換されず、そのまま貼り付けられます。貼り付け後に履歴へ記録される場合は、展開後のテキストが履歴に残ります。

English

Snippet variables are expanded only when a registered snippet is pasted. Regular clipboard history paste operations do not expand variables.

Custom date and time formats are supported with either colon or function-style syntax.

{{date("yyyyMMdd")}}
